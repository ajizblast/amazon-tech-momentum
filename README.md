# TechMomentum

TechMomentum is a modern, responsive web application that showcases the "Top 10" latest tech products across various categories like Gaming Laptops, Smart Home devices, and Mobile Gaming. It features a sleek UI with dark/light mode toggle built using React and Tailwind CSS.

## Tech Stack

*   **Framework**: [React](https://react.dev/) + [Vite](https://vitejs.dev/)
*   **Styling**: [Tailwind CSS v3](https://tailwindcss.com/)
*   **Components**: Custom React components inspired by modern UI patterns
*   **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

To run the project locally:

1.  **Clone the repository** (if you haven't already):
    ```bash
    git clone <repository-url>
    cd amazon-top-10/app
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

3.  **Start the development server**:
    ```bash
    npm run dev
    ```

4.  **Build for production**:
    ```bash
    npm run build
    ```

## Features

*   **Responsive Layout**: Optimized for both mobile and desktop screens using Tailwind's container queries.
*   **Dark/Light Mode**: Integrated theme switcher that respects system preferences and saves user choice in local storage.
*   **Modern Aesthetics**: Uses Inter and Noto Sans fonts with premium design elements and hover effects.

## Future Roadmap

This project is actively being developed. Upcoming features include:
